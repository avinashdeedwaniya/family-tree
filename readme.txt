== Description ==

WP Family Tree is a family tree generator plugin for WordPress.

Family and Members are custom post type where you can manage family image then member information like: name, DOB, member info etc.

A nice graphic representation will come on family detail page.

A list of all families will show on any page by using [family_listing].

Single user can create own family after login in wordpress. He can create 1 family from one account.

Any suggestions for future enhancements welcome.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
